Fellini's Pizzeria

Filter Page

React App Development

Brief

The goal of this project was to build my own live filtering page based on appropriate design heuristics. Wanting to create something personally meaningful and useful, I decided to go with the Pizza place that's close to many Brown students' hearts - Fellini's Pizzeria down on Wickendown Street. As great as its pizzas are, the pizzeria unfortunately does not present the ingredients and toppings of its pizzas clearly on the menu in its restaurant, making dietary selection tricky at times. I am particularly excited about this project, as it will be my very first time exploring the React library, knowing how powerful it is.

Ideation and Wireframing

To maximise the usability of the web page, my priority was to keep its components simple. After discussing with my peers and completing a competitive analysis of well-designed websites like YouTube, Amazon and Netflix, I decided on the following design choices:

  1. Using a card layout to contain the information of each pizza item.
  2. Making the user filter selection explicit and reversible, ie. the user can clear the filter if desired.
  3. Adding a price sorting function.

Design

Sticking to the initial design goals, I kept the layout simple and alllowed the user to switch between filters using radio buttons. I also want to provide more pizza ingredient details to the user, so I created a hover effect that converts the images into recipe text in real-time.

Challenges

I initially found it challenging to pass multiple filtering functions to the same Component that holds my pizza array, but managed to get it to work using Javascript built-in map() and filter() functions.

Redesign

After receiving some peer critiques, I would like to implement a couple of changes to make the interface more usable:

  1. Creating a drop-down menu for each filters, so that more space can be dedicated to the main pizza container.
  2. Implementing an F-layout, such that the filters are presented in a side bar so that the user doesn't have to scroll back up to change filter settings.

Reflection

Overall, I really enjoyed the process of coding in React. It not only makes my application fast, responsive and user-friendly, but also so much easier to maintain over time. I am excited to build more layers unto this project and explore React for mobile app development.
